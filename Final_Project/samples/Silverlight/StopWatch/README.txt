StopWatch Sample Requires Silverlight 4 SDK.
http://www.silverlight.net/

For Beta, this sample merely demonstrates how to add and use CCR for Silverlight inside of a Silverlight Application.

To run or debug,
-Add a new Web Application/Site which hosts this sample.
-Open the properties of the new Application/Site, on the Silverlight Tab - click Add and select this sample project.
-Set the Web Application/Site as the startup project.
-Set the test page for the Silverlight Application as the start page.
